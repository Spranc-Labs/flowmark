What is Flowmark?

Flowmark is a zero-dependency highlighting library that flows across element boundaries. Unlike traditional highlighting solutions, Flowmark allows you to select and highlight text that spans multiple DOM elements seamlessly.

Key Features

Cross-element highlighting: Select text across multiple paragraphs, headings, and other elements. Flowmark intelligently wraps your selection with highlight marks that preserve the document structure.

Precise text matching: Using context-aware text positioning with prefix and suffix validation, Flowmark can restore your highlights even if the page content changes slightly.

Storage flexibility: Use LocalStorage for browser persistence, PostMessage for iframe communication, or create custom adapters for your backend.

Getting Started

Installing Flowmark is simple: npm install @spranclabs/flowmark Then import the Highlighter class and a storage adapter. Initialize with your container element and configuration options. That's it!
